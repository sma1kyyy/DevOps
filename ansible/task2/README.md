Задача 2. 

ТЗ: Написать свой модуль который (на выбор):
• принимает в качестве параметра порт для nginx и подставляет его в конфигурацию
Формат сдачи: репозиторий в гите

Использовался яп python для написания модуля, библиотека ansible.module_utils.basic.AnsibleModule стандарт ansible, re.sub для поиска и замены порта в файле конфига. Файлы: library/nginx_port.py кастомный модуль, main.yaml плейбук.

Инструкция: 
Убедитесь, что установлен ansible, также, что модуль лежит в папке library/.
Запустите плейбук: 
ansible-playbook -i inventory.ini main.yaml
Проверьте доступность: 
http://192.168.31.171:8085/

или 

curl -I http://192.168.31.171:8085/
